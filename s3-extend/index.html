Installing The Python Extension Servers

For Windows

Open a command window and type:

pip install s3-extend

For Mac And Linux (including Raspberry Pi)

Open a command window and type:

sudo pip3 install s3-extend



It may take several minutes to download and install the necessary Python packages to support the Arduino, Raspberry Pi, and ESP-8266.
